# 7주차 실습 기록

## 계획

### 밸런스 패치

- 체력을 3에서 5로 늘릴 것 o
- 1단계 적의 공격 쿨타임을 3~5초에서 1~8초로 할 것 o

### 적용할 연출 기법

- 화면 흔들기: 체력 감소 시, 화면이 흔들리게 o
- 패럴랙스 스크롤링: 배경 화면에 여러 별들이 다른 속도로 지나가게 o
    - https://deep-fold.itch.io/space-background-generator에서 부분 추출할 것 o

### 화면

- 창 모드가 아닌, 전체 화면이 가능하게 할 것: 800x600에 맞게 나머지 부분은 레터 박스로 채우기 o
- 시작 화면 만들기. 난이도 선택 추가

### 분량

- 3스테이지까지 만들며, 각 스테이지마다 새로운 적 하나를 넣을 것

### 기타

- 적 생성 시 한 쪽부터 채우는 것이 아닌, 양쪽을 같이 채우기 o
- 적이 죽은 후에도 공격하는 버그 해결할 것 o
- 폴더를 만들어서 이미지, 사운드를 관리할 것 o
- 미사일 함선 폭발 시 이미지, 피격 시 이미지, 체력바 o
